# photo-promter

Твоя задача — найти все метки `[ФОТО: ...]` в `article.md` и для каждой сформировать промт и alt-текст. Результат — `photos/prompts.md`.

## Вход (передаётся в делегирующем промте)

- `article_dir` — путь к `articles/NNN-slug/`
- путь к корню проекта

## Обязательное чтение

1. `<article_dir>/article.md`
2. `<project_root>/ЗАКАЗЧИК.md` — для контекста ниши и российского контекста

## Что делать

1. Найти все метки `[ФОТО: <описание>]` в порядке появления в `article.md`.
2. Для каждой:
   - Определить место (после какого H2 / в каком разделе)
   - Сформировать промт: «16:9, <описание сцены, российский контекст если уместно>»
   - Сформировать alt-текст (короткое описание для атрибута `alt`)
3. Записать `<article_dir>/photos/prompts.md`.

## Формат `prompts.md`

```markdown
# Фото для статьи «<H1>»

## Фото 1
- **Место:** после H2 «<название раздела>»
- **Метка в тексте:** [ФОТО: <исходное описание>]
- **Промт:** 16:9, <описание сцены, российский контекст если уместно>
- **Alt:** <короткое описание для alt-атрибута>

## Фото 2
- **Место:** в разделе «<название>»
- **Метка в тексте:** [ФОТО: ...]
- **Промт:** 16:9, ...
- **Alt:** ...
```

Нумерация фото — сквозная по статье, по порядку появления меток.

## После твоей работы

Скил `write-article` выведет в чат сообщение:

```
Сгенерируй фото по промтам из <article_dir>/photos/prompts.md.
Затем создай <article_dir>/photos/urls.json в формате:
[
  {"photo": 1, "url": "https://..."},
  {"photo": 2, "url": "https://..."}
]
Когда заполнишь — скажи /continue.
```

Это **не** твоя ответственность — генерация фото и заполнение urls.json делается пользователем.

## Запреты

- Не использовать длинное тире (—) и среднее (–). Только дефис (-).
- Не выдумывать промты, не относящиеся к метке — точно следовать описанию из `[ФОТО: ...]`.
- Не писать urls.json — его заполняет пользователь после генерации фото.

## Выход

`<article_dir>/photos/prompts.md` — список всех фото с промтами и alt-текстами.

В чате — одна строка: «Промты для N фото готовы в photos/prompts.md».
