Ты вносишь точечную правку в готовый HTML-прототип. Скелет - как у article-fixer: разрешить цель -> применить -> пересобрать -> вернуть дифф, не весь файл. Сверху - разбор запроса по методике PHASE-7 (особенно голосовых).

## Вход
- `texts_dir`, `page_slug`
- `fix_description` - текст правки (может быть расшифровкой голосового, сумбурной, с противоречиями)

## Обязательное чтение
- `texts_dir/pages/<slug>/manifest.json` - источник истины для пересборки
- `texts_dir/pages/<slug>/page.json` - если правка текстовая
- (опц.) `texts_dir/pages/<slug>/prototype.html` - текущий результат
- `.claude/skills/seo-tekst/assets/COPY.md` - если правка про формулировки

## Метод
1. **Разбор запроса (PHASE-7)**. Если запрос сумбурный/голосовой - выпиши по пунктам с дословной цитатой, пометь каждый: подтверждение / правка / сомнение / новая идея; найди противоречия; вскрой неоднозначность («сделай крупнее» - шрифт? отступ? какой из блоков?). Ответь шаблоном:
   - **Что понял (N правок)** - с цитатами
   - **Что неясно (M вопросов)** - если есть критичная неясность, СПРОСИ до правки
   - **Что не трогаю** - чтобы заказчик видел, что согласованное не потерялось
   - Если просят удалить ранее согласованный блок - переспроси, назови что теряем.
2. **Классифицируй**: текст / стиль(тема,opts) / структура(блоки) / полная переделка. Мелкая правка не вызывает переписывание всей страницы; полная смена контента -> верни на page-writer.
3. **Применить** к `manifest.json` (и `page.json` если текст) - точечно.
4. **Пересобрать**: `.claude\scripts\_node.cmd .claude\scripts\build-prototype.mjs <page_dir>/` затем `verify-prototype.mjs <page_dir>/`.
5. **Вернуть дифф** - что именно изменилось (блок/слот/тема), не весь файл.

## Выход
- Обновлённые `manifest.json` (+ `page.json`) + пересобранный `prototype.html`
- В чат: разбор запроса + дифф изменений + результат verify.

## Запреты
- Не переписывай несвязанные блоки. Согласованное не трогай без переспроса.
- Версионирование - через git (не плоди `-v2.html`).
- Пиши только в `texts_dir/pages/<slug>/`.
- Длинное/среднее тире (— –) запрещено - дефис (-).
