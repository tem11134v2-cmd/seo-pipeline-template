## Дизайн и CSS-переменные

Токены синхронизированы с утверждённой статьёй-эталоном (см. `template.html`). В вёрстке статей блога роли цветов: **оранжевый `#f07f2f` - декоративный акцент** (маркеры списков, скроллбары таблиц, маркеры FAQ, кнопка CTA), **синий `#0080ff` - ссылки**, **тёмно-синий `#181d3b` - текст и заголовки**. Макет плоский (без карточки), фон белый.

| CSS-переменная | Назначение | Значение |
|----------------|-----------|---------|
| `--nx-dark` | Основной текст и заголовки | #181d3b |
| `--nx-gray` | Вторичный/приглушённый текст | #5a6178 |
| `--nx-accent` | Акцент (маркеры, скроллбары, кнопки, FAQ) | #f07f2f |
| `--nx-accent-hover` | Hover акцента | #d96a1a |
| `--nx-link` | Ссылки | #0080ff |
| `--nx-link-hover` | Hover ссылок | #0066ce |
| `--nx-bg` | Фон страницы | #ffffff |
| `--nx-white` | Белый | #ffffff |
| `--nx-light` | Светлый фон (блоки, таблицы, цитаты) | #f5f5f5 |
| `--nx-border` | Бордеры | #e7e7e7 |
| `--nx-radius` | Скругление крупное | 20px |
| `--nx-radius-sm` | Скругление мелкое | 10px |
| `--nx-radius-pill` | Пилюли (теги, кнопки) | 3000px |
| `--nx-font` | Шрифт | Cygre, Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Arial, sans-serif |

> Шрифт: Cygre (фирменный) с fallback Inter -> системный sans-serif.
> Компоненты статьи: хлебные крошки, H1 + дата, оглавление (сворачиваемое), таблицы (с горизонтальным скроллом), figure, blockquote, блок-диаграмма `.nx-chart` (инлайн-SVG), CTA-блок `.nx-cta`, плитка тегов, FAQ (`<details>/<summary>`), блок автора.
> Главный сайт traffic-masters.ru сделан на Next.js с другой палитрой (там primary - синий #0080ff). Для блога/статей действует палитра эталонной статьи выше (оранжевый акцент).

## Платформа и хостинг

| Поле | Значение |
|------|---------|
| Платформа (основной сайт) | Next.js (React, Turbopack, Tailwind v4; не классическая CMS) |
| Платформа блога/статей | Tilda (статьи публикуются HTML-блоком; в `template.html` сохранены `#allrecords`-оверрайды; изображения на static.tildacdn.com) |
| URL блога | /blog |
| Шаблон вёрстки | template.html (синхронизирован со статьёй-эталоном) |
| Особенности вёрстки | Статья - самодостаточный HTML (инлайн-CSS + инлайн-SVG, без внешних CDN/иконок). Маркер вставки тела - `<!-- CONTENT -->`. Schema.org (Article + FAQPage + BreadcrumbList) добавляет сборщик из meta.json |

> Важно: профиль изначально предполагал блог на Next.js (/blog/ 404). Статья-эталон показала, что фактически статьи идут через Tilda (tildacdn-изображения, `#allrecords`). Подтвердить у клиента финальную площадку публикации.
