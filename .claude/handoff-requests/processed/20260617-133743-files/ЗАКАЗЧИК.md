## Дизайн и CSS-переменные

Цвета, шрифты и скругления. Значения взяты из CSS сайта заказчика (Tailwind v4 theme tokens, чанк `/_next/static/chunks/*.css`).

| CSS-переменная | Назначение | Значение |
|----------------|-----------|---------|
| `--nx-dark` | Основной текст | #1C1A17 (тёплый near-black, токен `--ink`) |
| `--nx-gray` | Вторичный текст | #9A8F80 (тёплый серо-бежевый, токен `--muted`) |
| `--nx-accent` | Акцентный цвет (ссылки, иконки, маркеры) | #BC6A4A (терракота/clay, токен `--clay`) |
| `--nx-accent-hover` | Hover акцента | #D98A66 (светлая терракота, токен `--clay-light`) |
| `--nx-bg` | Фон страницы | #F6F1E9 (тёплый paper / off-white, токен `--paper`) |
| `--nx-white` | Белый | #FFFFFF |
| `--nx-light` | Светлый фон (блоки, таблицы) | #F6F1E9 (paper; для контраста блоков можно #FFFFFF) |
| `--nx-border` | Бордеры | rgba(0,0,0,0.09) (токен `--line: #00000017`) |
| `--nx-radius` | Скругление (основное) | 12px (Tailwind `rounded-xl` = 0.75rem) |
| `--nx-radius-sm` | Скругление (мелкие элементы) | 8px (Tailwind `rounded-lg` = 0.5rem) |
| `--nx-font` | Шрифт | Hanken Grotesk (основной текст), system-ui, -apple-system, sans-serif |

> Заголовки на сайте - Bricolage Grotesque (display), токен `--font-bricolage`. Тело текста - Hanken Grotesk, токен `--font-hanken`.
> Палитра тёплая, "журнальная": бумажный фон + чернильный текст + терракотовый акцент. Не использовать холодные/яркие SaaS-цвета.

## Платформа и хостинг

| Поле | Значение |
|------|---------|
| Платформа | Свой хостинг, headless-фронтенд |
| CMS (если есть) | Next.js (App Router, RSC, Turbopack) + Tailwind CSS v4. SSR/SSG. Источник контента (CMS/headless) не определён |
| URL блога | /blog/ (решение заказчика). Статьи лежат в /blog/<slug>. Раздел пока не опубликован - путь согласован под будущую структуру |
| Шаблон вёрстки | template.html (готовит template-designer отдельно) |
| Особенности вёрстки | Next.js SPA: серверный рендер главной отдаёт контент в RSC-payload; пробные пути дают 404-shell. robots.txt и sitemap.xml отсутствуют (404). meta robots на отданных страницах - noindex (проект, вероятно, ещё в разработке/закрыт от индексации) |

> URL блога по умолчанию `/blog/`. Для существующих сайтов — указать текущий URL раздела блога.
> Используется в BreadcrumbList (Schema.org) и хлебных крошках.
> ⚠️ Сейчас на отданных страницах стоит `<meta name="robots" content="noindex">` - перед запуском SEO-публикаций проверить, что индексация открыта.
