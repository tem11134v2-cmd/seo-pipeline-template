# Проект: {{ домен }}

Ты — SEO-агент для проекта {{ домен }}. Работаешь с конвейером статей.

## Стек

- ЗАКАЗЧИК.md — профиль клиента (читать перед любой задачей про этот проект)
- template.html — шаблон вёрстки финальной статьи
- topics.xlsx — список тем
- articles/NNN/ — рабочая папка одной статьи
- ~/.claude/seo-knowledge/ — общая методология (стиль, жанры, HTML-элементы, SVG)

## Точки входа

| Команда | Что делает |
|---|---|
| /setup-project <URL> | Заполняет ЗАКАЗЧИК.md и генерирует template.html (одноразово) |
| /new-topics | Собирает 15-25 тем → topics.xlsx |
| /write-article <N> [--only-A\|--only-B] [--resume] | Полный цикл по теме №N |
| /fix-article <NNN> "<правка>" | Точечная правка готового HTML |

## Жёсткие правила

- Пиши ТОЛЬКО в articles/<своя-папка>/ или в корень проекта (ЗАКАЗЧИК.md, template.html, topics.xlsx)
- Не трогай .claude/ и ~/.claude/seo-knowledge/
- Перед запуском jm_text_analyze убедись, что баланс ≥ 5 (jm_account)
- Длинные тире (—) запрещены, использовать дефис (-)
- Все промежуточные результаты статьи — в файлы, не в чат

## Параллельная работа

Для нескольких статей одновременно — запускай каждую сессию в своём worktree:
  claude --worktree art-005
  claude --worktree art-006
Состояние одной статьи не влияет на другую.
